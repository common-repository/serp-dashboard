Serp Dashboard helps you track your blogs rank in the search engine.

== Description ==

Serp Dashboard takes your 3 top blog tags, makes any combination and queries Google. 

You can add a custom search and schedule your searches. 

features :

* standard 3 tag permutation
* queries google in any language
* enter custom searches
* schedule searches
* jquery ajax admin page
* automated (uses ’shutdown’ hook to trigger searches)
* uses its own tables
* preconfigured, runs on activation
* clean uninstall on deactivation


== Installation ==


1. Upload the directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Serp Dashboard from the Settings menu in Wordpress
